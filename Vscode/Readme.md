## 📦 Repository Overview

This repository (`Code - OSS`) is the open-source foundation of **[Visual Studio Code](https://code.visualstudio.com)**.  
Here, Microsoft and the community collaborate on:

- Core source code and issue tracking  
- [Roadmap](https://github.com/microsoft/vscode/wiki/Roadmap) and [monthly iteration plans](https://github.com/microsoft/vscode/wiki/Iteration-Plans)  
- [Endgame plans](https://github.com/microsoft/vscode/wiki/Running-the-Endgame)

**Visual Studio Code** is a distribution of `Code - OSS` with Microsoft-specific customizations, released under the [Microsoft product license](https://code.visualstudio.com/License/).

### Highlights
- Lightweight editor with full **edit–build–debug** cycle support  
- Rich language features, navigation, and debugging  
- Extensible ecosystem with thousands of extensions  
- Seamless integration with existing tools  

👉 Download for [Windows, macOS, and Linux](https://code.visualstudio.com/Download)  
👉 Try the [Insiders build](https://code.visualstudio.com/insiders) for daily updates

## 📦 Bundled Extensions

VS Code ships with built-in extensions in the [extensions](extensions) folder:  
- Language grammars and snippets  
- Rich language features (`*-language-features` suffix)  

Example:  
- `json` → syntax coloring  
- `json-language-features` → navigation, suggestions

## 🐳 Development Container

This repo includes Dev Containers and GitHub Codespaces support.

- **Dev Containers**: Use **Clone Repository in Container Volume...** for optimized disk I/O.  
- **Codespaces**: Install the [Codespaces extension](https://marketplace.visualstudio.com/items?itemName=GitHub.codespaces) and create a new codespace.  

Requirements: **4 cores, 6 GB RAM (8 GB recommended)**  
See [.devcontainer/README.md](.devcontainer/README.md) for details.
